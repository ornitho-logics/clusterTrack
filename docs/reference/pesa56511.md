# ARGOS satellite tracking data for an individual Pectoral Sandpiper

A dataset containing ARGOS locations for an individual male Pectoral
Sandpiper

## Usage

``` r
pesa56511
```

## Format

A data.table with 882 rows and 4 columns:

- latitude:

  Numeric. Latitude in decimal degrees (WGS84).

- longitude:

  Numeric. Longitude in decimal degrees (WGS84).

- locationDate:

  POSIXct. Timestamp of location fix (UTC).

- locationClass:

  Character. ARGOS location quality class (e.g., B, 0, A).

## Source

Kempenaers, Bart, and Mihai Valcu. "Breeding site sampling across the
Arctic by individual males of a polygynous shorebird." Nature 541.7638
(2017): 528-531.
