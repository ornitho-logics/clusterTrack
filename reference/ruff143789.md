# ARGOS satellite tracking data for an individual Ruff

A dataset containing ARGOS locations for an individual male Ruff.

## Usage

``` r
ruff143789
```

## Format

For `ruff143789`: a data.table with 2834 rows and 4 columns.

## Source

Kempenaers, Bart, et al. "Large-scale sampling of potential breeding
sites in male ruffs." Proceedings B 292.2038 (2025): 20242225.
https://royalsocietypublishing.org/doi/full/10.1098/rspb.2024.2225.

## Details

`ruff143789` is the full dataset; `mini_ruff` is a reduced subset used
for examples and tests.

## Examples

``` r
data(ruff143789)
```
